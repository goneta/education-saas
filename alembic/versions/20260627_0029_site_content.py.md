# 20260627_0029_site_content.py

## Purpose

- Creates the `site_content` table backing the public Site TeducAI CMS managed by the Super Admin.

## Local Contracts

- Single JSON `data` column holds the editable public content document; `updated_by_id` references the Super Admin who last saved.
- Additive and reversible: the public site falls back to code-level defaults when the table is empty, so no data backfill is required.

## Verification

- `python -m alembic heads`
- `python -m py_compile backend\models.py`
