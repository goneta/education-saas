# my-assignments/page.tsx — Student/parent homework page

Student/parent UI over /assignments/mine: to-do / submitted / graded columns;
opening an assignment shows the questions (MCQ radios / textareas), autosave +
submit (online mode), the teacher feedback and score, and the answer key when
released. Parents get a child selector (via /self-documents/children), read-only.
Copy via useTranslations("assignments") (4-locale).
